🖼️ 识图题 1 电动操动机构隔离开关
📌 题目
依据应用CL2型电动操动机构隔离开关的控制信号回路，分析其所控制的隔离开关手动分闸控制过程。
⚠️ 图片加载失败
✅ 参考答案（分步解析）：
  1. 发出分闸指令：将控制开关“2SA”转至分闸位，触点2SA2-4闭合，分闸接触器线圈（KMO）得电：
    +WC → 1FU → 1KTP4-12 → 1SA2-1 → 2SA2-4 → KMO5-6 → KMC11-12 → 2ST1-2 → 3ST1-2 → 2FU → -WC。
  2. 接触器自保持：KMO得电后，辅助触点KMO7-8闭合，实现自保持。
  3. 电机通电旋转：KMO主触点KMO1-2、KMO3-4闭合，接通电动机回路：
    +WO → 3FU → QM1-2 → 电机转子 → KMO1-2 → 励磁绕组W1-4 → KMO3-4 → QM3-4 → 4FU → -WO。
    电动机逆时针旋转，驱动隔离开关分闸。
  4. 分闸到位停止：分闸到位后，行程开关2ST1-2断开，KMO失电，电机停转。
  5. 位置信号：辅助接点QS17-19闭合，分闸位置继电器KTP得电，KTP11-9闭合，绿灯HLG亮起，表示分闸位。
🖼️ 识图题 2 馈线侧电气主接线
📌 题目
如图所示为某牵引变电所馈线侧电气主接线图，依据该图写出A相电容投入运行倒闸操作步骤。
⚠️ 图片加载失败
✅ 参考答案：
  1. 确认A相电容的接地线已经全部撤除。
  2. 确认A相电容的接地封线已经全部撤除。
  3. 确认A相电容已无其他接地。
  4. 合上2411QS（隔离开关）。
  5. 确认2411QS处在合闸位。
  6. 推上241QF小车至运行位止滑。
  7. 闭锁分闸门。
  8. 合上241QF（断路器）。
  9. 确认241QF处在合闸位。
